Steps to make a Portfolio

Step one

Head to this github and fork it to your project

Step two

Once you have the project forked, then you got open up the terminal in the project and type "npm install". This will install all the necessary dependancies required to run the project.

Step three

When all the dependancies are installed, then you can begin changing the project to what you want to display on your portfolio.

There you have it. How to make a quick portfolio.
